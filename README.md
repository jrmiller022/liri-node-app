# liri-node-app

Make it so liri.js can take in one of the following commands:

   * `concert-this` (Bands name or Artist)
      - Venue Name
      - Location of show
      - Date of show

   * `spotify-this-song` (Song)
      - Artist
      - Song title
      - Album
      - URL

   * `movie-this` (movie title)
      - Title
      - Year made
      - IMDB rating 
      - Country produced
      - Language 
      - Plot
      - All actors starred

   * `do-what-it-says`
      - Artist
      - Song
      - Album  
      - URL
